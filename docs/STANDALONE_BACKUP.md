# Standalone版本备份说明

## 备份信息

- **备份分支**: `backup-standalone`
- **备份标签**: `v-standalone-final`
- **备份提交**: `5785430` - 修复ISR.c中的格式问题
- **备份日期**: 2026-01-29

## 当前状态

此版本为ZYNQ Standalone（裸机）模式的最终稳定版本，包含以下功能：

### 已实现功能
- ✅ 网络初始化（固定IP: 192.168.0.123/24）
- ✅ 网络监测（每5秒检测链路状态）
- ✅ 异步网络初始化，不阻塞示波器启动
- ✅ 网络监测在10ms定时器中断中运行

### 已知问题
- ⚠️ 链路状态检测可能不准确（链路状态一直显示DOWN）
- ⚠️ 无法在主循环中处理网络输入（会导致触摸响应问题）
- ⚠️ Standalone模式不支持多线程，网络处理受限

## 迁移计划

**准备迁移到Linux系统**，原因：
1. Linux内核提供完整的网络驱动和PHY管理
2. 支持多线程，网络处理不会阻塞UI
3. 可以使用标准的网络工具和配置
4. 更好的硬件抽象和驱动支持

## 分支说明

- `master`: 当前开发分支（准备迁移到Linux）
- `backup-standalone`: Standalone版本备份分支（暂时不再维护）

## 恢复方法

如果需要恢复到此版本：

```bash
# 切换到备份分支
git checkout backup-standalone

# 或使用标签
git checkout v-standalone-final
```

## 注意事项

⚠️ **此版本暂时不再维护**，所有新功能开发将在Linux版本中进行。
